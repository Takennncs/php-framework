# Takenncs Framework

Takenncs Framework - Modern PHP Framework
- Your custom PHP framework is running successfully!

## ✨ Features

- MVC Architecture
- Clean Routing System
- Blade-like PHP Views
- TailwindCSS UI
- Dark / Light mode
- Component-based layout
- Simple and lightweight

## 📦 Installation

Clone the repository:

git clone https://github.com/Takennncs/php-framework.git

Install dependencies:

composer install

Run locally:

php -S localhost:8000 -t public
